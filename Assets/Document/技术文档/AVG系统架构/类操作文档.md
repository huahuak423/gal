## 二、核心类详解

### 2.1 PlayerDataComponent

**文件位置**：`Assets/GameMain/Scripts/Base/PlayerDataComponent.cs`

**命名空间**：`GameMain.Scripts.Base`

**继承自**：`GameFrameworkComponent`

**职责**：存储和管理玩家的所有运行时数据

#### 属性列表

| 属性 | 类型 | 说明 |
|-----|------|------|
| `Charm` | int | 魅力值 |
| `Inspiration` | int | 灵感值 |
| `Sanity` | int | 理智值 |
| `CurrentActionPoints` | int | 当前行动点（只读） |
| `MaxActionPoints` | int | 最大行动点（只读） |
| `CurrentRound` | int | 当前周目数（只读） |
| `BonusActionPoints` | int | 周目继承的行动点加成 |
| `BonusCharm` | int | 周目继承的魅力加成 |
| `BonusInspiration` | int | 周目继承的灵感加成 |
| `BonusSanity` | int | 周目继承的理智加成 |

#### 方法列表

##### 初始化方法

```csharp
// 初始化新周目的数据（清空好感度、物品，重置行动点）
void InitializeNewRound()

// 重置游戏（从主菜单开始新游戏时调用）
void ResetGame()
```

##### 属性操作

```csharp
// 增加玩家属性
void AddAttribute(PlayerAttributeType type, int value)

// 获取玩家属性值
int GetAttribute(PlayerAttributeType type)
```

##### 行动点操作

```csharp
// 消耗行动点（返回是否成功）
bool ConsumeActionPoints(int amount)

// 检查是否有足够的行动点
bool HasEnoughActionPoints(int amount)
```

##### NPC好感度操作

```csharp
// 获取NPC好感度
int GetFavorability(int npcId)

// 增加NPC好感度
void AddFavorability(int npcId, int value)

// 设置NPC好感度
void SetFavorability(int npcId, int value)
```

##### 物品操作

```csharp
// 添加物品
void AddItem(string itemId)

// 移除物品
bool RemoveItem(string itemId)

// 检查是否拥有物品
bool HasItem(string itemId)
```

##### 条件检查

```csharp
// 检查单个条件是否满足
bool CheckCondition(ChoiceCondition condition)

// 检查多个条件是否全部满足
bool CheckConditions(List<ChoiceCondition> conditions)
```

##### 奖励应用

```csharp
// 应用单个奖励
void ApplyReward(ChoiceReward reward)

// 应用多个奖励
void ApplyRewards(List<ChoiceReward> rewards)
```

##### 周目操作

```csharp
// 结束当前周目（计算继承加成）
void EndRound()

// 开始新周目
void StartNewRound()
```

#### 使用示例

```csharp
// 获取 PlayerDataComponent
var playerData = CustomEntry.PlayerData;

// 消耗行动点
if (playerData.ConsumeActionPoints(1))
{
    // 行动点扣除成功，执行行动
}

// 增加属性
playerData.AddAttribute(PlayerAttributeType.Charm, 5);

// 增加NPC好感度
playerData.AddFavorability(1001, 10);

// 检查是否拥有物品
if (playerData.HasItem("key_item_001"))
{
    // 拥有该物品
}

// 开始新周目
playerData.StartNewRound();
```

### 2.3 ActionPointChangedEventArgs

**文件位置**：`Assets/GameMain/Scripts/AVGSystem/Runtime/ActionPointChangedEventArgs.cs`

**命名空间**：`AVGGame.Runtime`

**继承自**：`GameEventArgs`（UGF 事件基类）

**职责**：行动点变化时触发的事件参数

#### 属性列表

| 属性 | 类型 | 说明 |
|-----|------|------|
| `Id` | int | 事件ID（重写自基类） |
| `CurrentActionPoints` | int | 当前行动点数 |

#### 方法列表

```csharp
// 创建事件参数（从引用池获取）
static ActionPointChangedEventArgs Create(int currentActionPoints)

// 清理（归还引用池时调用）
override void Clear()
```

#### 使用示例

```csharp
// 订阅事件
GameEntry.Event.Subscribe(ActionPointChangedEventArgs.EventId, OnActionPointChanged);

// 触发事件
GameEntry.Event.Fire(this, ActionPointChangedEventArgs.Create(newActionPoints));

// 处理事件
private void OnActionPointChanged(object sender, GameEventArgs e)
{
    var args = (ActionPointChangedEventArgs)e;
    Debug.Log($"行动点变化: {args.CurrentActionPoints}");
}

// 取消订阅
GameEntry.Event.Unsubscribe(ActionPointChangedEventArgs.EventId, OnActionPointChanged);
```

## 三、流程类详解

### 3.1 ProcedureMap

**文件位置**：`Assets/GameMain/Scripts/Procedure/ProcedureMap.cs`

**命名空间**：`AVGGame.Procedure`

**继承自**：`ProcedureBase`

**职责**：管理大地图流程，显示可选地点供玩家选择

#### 流程生命周期

| 方法 | 说明 |
|-----|------|
| `OnEnter` | 订阅事件，检查行动点，打开大地图UI |
| `OnLeave` | 取消订阅，关闭UI |
| `OnUpdate` | 每帧更新（大地图通常不需要） |

#### 公共方法（供UI调用）

```csharp
// 选择地点 - 由 MapPanel 调用
void SelectLocation(int locationId)

// 获取当前行动点 - 供UI显示
int GetCurrentActionPoints()

// 获取当前周目 - 供UI显示
int GetCurrentRound()
```

#### 流程切换

```
进入 ProcedureMap
       │
       │ 玩家选择地点
       ▼
  ProcedureLocation
```

#### 使用示例

```csharp
// 在 UI 中调用
public void OnLocationButtonClick(int locationId)
{
    // 获取当前 ProcedureMap（通过 UI 打开时传入的 userData）
    var procedureMap = userData as ProcedureMap;
    procedureMap.SelectLocation(locationId);
}
```

### 示例3：订阅行动点变化事件

```csharp
// 在某个 UI 类中订阅事件
protected override void OnOpen(object userData)
{
    base.OnOpen(userData);
    GameEntry.Event.Subscribe(ActionPointChangedEventArgs.EventId, OnActionPointChanged);
}

protected override void OnClose(bool isShutdown, object userData)
{
    base.OnClose(isShutdown, userData);
    GameEntry.Event.Unsubscribe(ActionPointChangedEventArgs.EventId, OnActionPointChanged);
}

private void OnActionPointChanged(object sender, GameEventArgs e)
```
